                    ❓ QUICK FAQs
═══════════════════════════════════════════════════════════════════════════════

Q: Can I customize the design?
A: Yes! Edit src/App.css for colors, layout, fonts

Q: Can I change the rate limit?
A: Yes! Edit src/ai-router.js line 20: change 5000 to desired milliseconds

Q: How do I add more AI models?
A: Edit src/ai-router.js MODEL_FALLBACK_CHAIN - add new model to array

Q: Can I disable a security layer?
A: Yes, but not recommended! Comment out in src/securitySentinel.js activate()

Q: How do I see security incidents?
A: Firebase Console → Firestore → securityIncidents collection

Q: How do I unlock admin panel if locked?
A: Check Telegram for OTP, enter it on unlock screen

Q: Can I run this on localhost only?
A: Yes! npm run dev will work locally without deployment

Q: Is this GDPR compliant?
A: Mostly yes, but add privacy policy before launch
