📌 EXACT CONNECTION FLOW
Browser HTTPS → CollectiveConsciousness.jsx
Telegram Webhooks → telegram-bridge/index.js
consensusGateway HTTP → /api/consensus route

📁 COMPLETE FILE CHAIN
src/pages/CollectiveConsciousness.jsx → src/services/gateways/consensusGateway.js
src/services/gateways/consensusGateway.js → bff/routes/consensusRoutes.mjs
bff/routes/consensusRoutes.mjs → bff/services/consensusEngine.mjs
bff/services/consensusEngine.mjs → ml-engine/inference/predictor.py
ml-engine/inference/predictor.py → ml-engine/models/direction/*.py
ml-engine/models/direction/*.py → ml-engine/inference/consensus_aggregator.py
consensus_aggregator.py → returns to BFF → Frontend
