🧠 BRAIN - Frontend (React)

The brain controls everything - it shows the UI, processes user clicks, and decides what to display. Just like your brain tells your body what to do.

src/pages/CollectiveConsciousness.jsx
src/features/consensus/
src/components/

❤️ HEART - BFF (Node.js)

The heart pumps blood to the whole body - the BFF pumps data to the whole app. It coordinates between frontend and ML engine, just like your heart keeps everything supplied.

bff/server.mjs
bff/routes/consensusRoutes.mjs
bff/services/consensusEngine.mjs

🫃 LIVER - ML Engine (Python)

The liver processes toxins and nutrients - ML Engine processes data and predictions. It digests candle data and produces trading signals like a liver processes food.

ml-engine/main.py
ml-engine/inference/predictor.py
ml-engine/models/direction/

💪 ARMS - External Services

Arms reach out to get things - External services reach out to APIs. Telegram sends messages, News API fetches data, just like arms grab food or tools.

telegram-bridge/index.js
bff/services/newsService.mjs
bff/services/breakingNewsService.mjs

🦴 BONES - Database

Bones give structure - Database gives structure to all data. Just like bones hold your body together, database holds all app data.

ml-engine/data/candle_db.py
Firebase Realtime DB

🩸 BLOOD - Redis Cache

Blood carries oxygen quickly - Redis carries data quickly. Just like blood delivers oxygen fast, Redis delivers cached data fast.

Redis Cache
ml-engine/infrastructure/performance.py

⚙️ Core Functions Explained

📡 useConsensus

Fetches trading consensus signals from ML Engine
Input:
symbol: "NIFTY", timeframe: "5m"
Output:
{signal, confidence, regime, rrr}
consensusGateway.js → consensusRoutes.mjs

🧠 aiRouter

Routes user requests to correct security layer
Input:
{userId, intent}
Output:
{route, securityLevel}
src/services/ai-router.js

🔐 Security

Rate limiting, JWT validation, CORS
Input:
request.headers, token
Output:
{allowed, userId}
bff/services/security.mjs

📊 getConsensusSignal

Aggregates ML model votes
Input:
candles[], features
Output:
{signal, confidence}
bff/services/consensusEngine.mjs

🤖 predictSignals

ML prediction endpoint
Input:
symbol, candles
Output:
{direction, regime}
ml-engine/inference/predictor.py

🗳️ ConsensusAggregator

Combines model predictions
Input:
[votes from models]
Output:
{signal, confidence}
ml-engine/inference/consensus_aggregator.py
